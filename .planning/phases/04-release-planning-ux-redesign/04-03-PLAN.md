## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser to release APIs | User actions mutate releases, sprint membership, feature membership, and allocation spans |
| Drag/drop controls to domain actions | Pointer and keyboard interactions must produce valid sprint-id allocations |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-10 | Tampering | UI sends invalid span endpoints | mitigate | API validates; UI clamps controls to release sprint list |
| T-04-11 | Information Integrity | Capacity warning hidden or styled as success | mitigate | Amber warning state in column headers and feature cards |
| T-04-12 | Denial of Service | Empty release/sprint state breaks timeline render | mitigate | Explicit empty states before timeline grid |
| T-04-13 | Accessibility | Span changes require mouse-only resize | mitigate | Keyboard-accessible buttons/selects for start/end changes |
</threat_model>
